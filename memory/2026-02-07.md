## OAuth Research & Infrastructure (9 AM)

### Root Cause Found
- Maton (and similar OAuth wrappers) run Google apps in "Testing" mode
- Testing mode = tokens expire after 7 days by design
- Production mode = long-lived tokens (need app verification)

### Solution: Nango (Self-Hosted)
- Open-source OAuth infrastructure for 600+ APIs
- Auto token refresh + rotation
- Free to self-host
- nango.dev

### Blocker: Virtualization
- Current machine is a VM ("George's Virtual Machine")
- Docker requires Linux VM which needs hardware virtualization
- Nested virtualization not supported → Docker won't work here
