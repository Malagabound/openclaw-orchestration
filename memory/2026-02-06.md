### Critical Infrastructure Learning: Browser > OAuth

**The Problem:** Maton API started failing with "Missing app connection" errors. OAuth tokens expire unpredictably (24 hours or less).

**The Solution:** Browser-based email access as PRIMARY method
- Browser sessions persist for WEEKS (not hours like OAuth)
- No re-auth dance every day
- Actually reliable for production use

**Fallback Chain Established:**
1. Browser (primary) - use `profile="openclaw"`
